# Hero
## Create
method: `GET`   
route: `/hero/create`   
Render view to create new hero.

## Delete
method: `PUT`   
route: `/hero/delete/1`   
Delete the hero.

## Edit
method: `GET`   
route: `/hero/edit/1`   
Render view to edit a hero.

## List
method: `GET`   
route: `/hero/index`   
List all heroes.

## Save
method: `POST`   
route: `/hero/save`   
Save the new hero.

## Show
method: `GET`   
route: `/hero/show/1`   
Show one hero.

## Update
method: `PUT`   
route: `/hero/update/1`   
Save the updated hero.
